# CrawlTGDD
# Giới thiệu
Spider Crawl Web lấy thông tin điện thoại trên TGDĐ sử dụng thư viện Scrapy.

# Cài đặt
1. Cài đặt Scrapy
```bash
pip install Scrapy
```
# Sử dụng
Thao tác với console:
1. Chạy mô trường ảo
```bash
venv\Scripts\activate
```
2. Chạy spider
```bash
cd crawlTGDD
```
3. Xuất ra file json
```bash
scrapy crawl ducanh -o tenfile.json
```
